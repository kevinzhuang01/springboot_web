# SimpleTaskAPI

This is me demoing a simple RESTful API for managing tasks, built with Spring Boot.

## Features

- CRUD operations for tasks
- Mark tasks as completed
- Sort tasks by priority
- Filter completed tasks
- More to come....

## Technologies

- Java
- Spring Boot
- Lombok

## Getting Started

### Prerequisites

- Java 21 or higher
- Maven

### Build and Run

```sh
mvn clean install
mvn spring-boot:run
```
